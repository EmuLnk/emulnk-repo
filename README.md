## Contents

```
consoles.json          # System definitions (id, port, idAddress, idSize)
index.json             # Theme catalog for the in-app gallery
profiles/*.json        # Game profiles with memory data points
themes/*/              # Theme folders (theme.json + HTML/CSS/JS)
```
